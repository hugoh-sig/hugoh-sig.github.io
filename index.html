Sobre Mim

Conheça minha trajetória e especialidades

Sou Engenheiro Florestal com Mestrado em Análise e Modelagem de Sistemas Ambientais. Minha jornada começou com a curiosidade sobre como a tecnologia pode nos ajudar a entender melhor o meio ambiente. Hoje, trabalho combinando dados de satélite, programação e análise espacial para criar soluções que façam a diferença na conservação ambiental.

Gosto de trabalhar com Google Earth Engine, Python e R para processar imagens de satélite e criar visualizações interativas. Também desenvolvo aplicações que tornam dados geoespaciais mais acessíveis para quem precisa tomar decisões ambientais importantes. Estou sempre aprendendo e buscando maneiras de aplicar a tecnologia para resolver desafios ambientais reais.

Soluções inovadoras em geoprocessamento e análise ambiental

Mapa Web Paraopeba

Uma aplicação que criei para visualizar os impactos ambientais após as enchentes no Rio Paraopeba. Utiliza dados de satélite para mostrar mudanças na vegetação, sedimentos e óxidos de ferro na região, ajudando a entender melhor os efeitos do desastre ambiental.

JavaScript Leaflet.js Landsat 8/9 NDVI/SAVI SIG Web
Demo Código

Painel Óxidos de Ferro

Um painel interativo desenvolvido em R Shiny para explorar dados de óxidos de ferro em solos. Permite visualizar e analisar as relações entre diferentes propriedades do solo de forma intuitiva, facilitando a interpretação dos dados para pesquisadores e estudantes.

R Shiny ggplot2 Análise Multivariada Geoquímica
Demo Código

Códigos Google Earth Engine

Uma coleção de códigos que desenvolvi para processar imagens de satélite no Google Earth Engine. Inclui ferramentas para calcular índices de vegetação, analisar mudanças na paisagem ao longo do tempo e estudar recursos hídricos. Útil para estudos ambientais e análise territorial.
